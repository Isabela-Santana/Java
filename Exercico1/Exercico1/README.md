Calculadora de Dados e Geometria em Java
Este é um projeto simples desenvolvido em Java que interage com o usuário através do console. O objetivo do programa é coletar informações pessoais e realizar cálculos matemáticos básicos, servindo como um exercício de lógica de programação e manipulação de entrada de dados.

🚀 Funcionalidades
O programa executa as seguintes operações sequencialmente:

Coleta de Dados Pessoais: Solicita nome, ano de nascimento e país de origem.

Cálculo de Idade: Utiliza a data atual do sistema para calcular e exibir a idade aproximada do usuário.

Geometria - Quadrado: Calcula a área de um quadrado dado o valor de um lado.

Geometria - Retângulo: Calcula a área de um retângulo baseada na altura e base informadas.

Diferença de Idade: Solicita o nome e a idade de duas pessoas e calcula a diferença entre elas.

Tecnologias Utilizadas
Linguagem: Java
Bibliotecas: java.util.Scanner (para entrada de dados) e java.time.OffsetDateTime (para obter o ano atual).
